This is a list of essential web development vocabulary.

Vocabulary List

  • HTML: Stands for HyperText Markup Language, the standard markup language for documents designed to be displayed in a web browser.
  • CSS: Stands for Cascading Style Sheets, used for describing the presentation of a document written in a markup language like HTML.
  • JavaScript: A high-level, interpreted programming language primarily used to make web pages interactive.
  • DOM: Stands for Document Object Model, a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.
  • Frontend: Refers to the user-facing part of a website or application, including everything the user sees and interacts with.
  • Backend: Refers to the server-side of a website or application, handling data storage, logic, and communication with databases.
